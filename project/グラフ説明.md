# グラフ説明一覧

## ★01_★: 需要関数の係数の可視化
**ファイル名**: `01_demand_function_coefficients.png`

**内容**: 
- 需要関数から推定された3つの係数を棒グラフで表示
- 所得弾力性（α）、価格弾力性（β）、税率弾力性（γ）の大きさと符号を視覚化
- 各係数の値が論文の数値と一致することを確認

**主な特徴**:
- **所得弾力性（α=0.9622）**: 1に極めて近い値であり、これは所得が1%増加するとガソリン需要が約0.96%増加することを意味する。つまり、経済成長がガソリン需要をほぼ比例的に押し上げることを示している。この値が1に近いことは、ガソリンが必需品でありながら、経済成長と密接に連動していることを示唆している。

- **価格弾力性（β=3.2116）**: 3を超える大きな正の値であり、価格が1%上昇すると需要が約3.21%減少することを意味する。この値の大きさは、長期的には価格変動に対して需要が比較的反応しやすいことを示している。ただし、本分析は年次データを用いているため、短期的な非弾力性（価格が上がってもすぐには消費量を減らせない）とは異なり、長期的には消費者が車両の燃費改善や代替交通手段への転換などにより需要を調整する余地があることを反映している。

- **税率弾力性（γ=3.2767）**: 価格弾力性（β=3.2116）とほぼ等しい値であり、税率が1%上昇すると需要が約3.28%減少することを意味する。価格弾力性と税率弾力性がほぼ等しいことは、消費者にとって価格上昇と税率上昇が需要に与える影響が同程度であることを示している。これは、ガソリン税制が需要構造に与える影響の大きさを裏付けており、税制設計において重要な示唆を与える。

## ★02_★: 消費者余剰の増分推移
**ファイル名**: `02_consumer_surplus_increase.png`

**内容**:
- 2008年から2025年までの各年の消費者余剰増分を時系列で表示
- 2008年暫定税率失効、2009年金融危機、2020年COVID-19パンデミックを縦線でマーカー

**主な特徴**:
- 2008年: -1.15兆円（暫定税率失効による価格上昇+16.4円/L）
- 2009年: +1.31兆円（金融危機による価格下落-35.8円/L）
- 2017年以降: 価格上昇に伴い余剰が減少傾向

## グラフの使用目的

これらのグラフは、以下の分析結果を視覚的に示すために作成されています：

1. **需要関数の推定結果**: 係数の大きさと符号を確認
2. **消費者余剰の定量化**: 時系列での変化と累積効果を可視化
3. **CPI寄与度分析**: 固定税率が価格変動に与える影響を確認
4. **政策イベントの影響**: 主要なイベントが価格と余剰に与えた影響を分解
